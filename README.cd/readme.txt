🛠️ Projeto CRM Genérico - Interface Profissional e Interativa

Este projeto é um CRM profissional, desenvolvido com Django como backend, e projetado para ser genérico, atendendo diversos nichos de mercado (comercial, serviços, saúde, educação, etc.). A interface será moderna, interativa e intuitiva, com foco na experiência do usuário final.

🎯 Requisitos do Sistema:

Backend robusto com Django (Python).

Interface frontend com:

sidebar.js para navegação lateral com animações.

Sistema de rotas internas bem estruturado.

Arquivos .html com integração ao sistema de templates do Django.

Uso de ícones modernos e recursos JavaScript para uma experiência interativa.

Paleta de cores principal:

Branco

Verde acizentado (ex: #A3B3A3)

Layout limpo, profissional e leve, adaptado a diferentes dispositivos.
